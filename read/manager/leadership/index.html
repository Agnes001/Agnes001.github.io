可复制的领导力

# 什么是可复制的领导力

  1. 领导力是可以学会的
  2. 每个人都具备领导力 “复制”的关键在于工具化。掌握工具，人人可以具备领导力，领导的实际效果也会事半功倍。
    工具化，把工作拆分成可以完成并检验的步骤。
  3. 员工的执行力等于领导的领导力
    • 第1遍，交代清除事项；
    • 第2遍，要求员工复述；
    • 第3遍，和员工探讨此事项的目的；
    • 第4遍，做应急预案；
    • 第5遍，要求员工提出个人见解。

# 领导和管理的区别

  1. 领导的核心驱动力 在于尊敬和信任,相信跟着你能成大事，才会真正投入到工作中，尽心尽力。 管理是用“怕”来驱动团队。
  2. 游戏改变领导力
    • 特征一：共同目标
      要给团队设置一个宏大的目标，使得打架集体向目标前进，形成凝聚力；
    • 特征二：及时反馈
      要对员工的行为做出灵活、准确的有效反馈，让员工有参与感，进而产生价值感；
    • 特征三：清晰明确的规则
      要为工作设置明确的规则，让所有人有规可循、有章可守，打造公平环境；
    • 特征四：自愿参与
      要尊重员工的意愿，培养员工参与工作的自觉意识，推动工作进程。

# 沟通视窗

  1. 什么是沟通视窗？

    自己知道 自己不知道
    公开象限 盲点象限 他人知道
    隐私象限 潜能象限 他人不知道

  2. 公开象限
    公开象限越大，名气越大

  3. 隐私象限

    1. 忘了说的秘密-“知识的诅咒”
    2. 不好意思开口，但知道的事情
    3. DDS（Deep Dark Secrets）

  4. 盲点象限
    自我认识中的黑暗地带，消除盲点象限，需要借助他人的帮助，由他人说出我们自己没有注意或者无法发现的盲点。

  5. 潜能象限
    每个人的潜能都值得去努力挖掘。
    即便明天是世界默认，今夜我也要在园中种满莲花。

  6. 象限间的互相转化
    自我揭示：隐私象限转化为公开象限
    恳求反馈：盲点象限转化为公开象限

提示

当公开象限不断放大，收获的便是尊敬和信任

# 管理者角色

领导者、管理者、执行者
  1. 领导者
    通过营造氛围来提升绩效的人
  2. 管理者
    通过管理他人来完成工作的人，培养出了什么样的队伍，不要凡事都亲历亲为，要给员工成长的空间。
  3. 执行者
    执行的核心定义是给出结果。

提示

优秀的执行者需要认真执行，给出结果，同时及时对工作内容本身进行反馈，形成自己的认识和见解。

# 领导力技术

  1. 倾听和提问是一门学问
    多提问开放式问题，“你觉得怎么样”，“你有什么想法”
  2. 反馈
    “干的不错”，“这个想法好”，给出理由。 学会反馈，员工更有干劲，团队也会更加和谐。

最有效的沟通是在第一时间直接给出答案，然后补充解释。
善于沟通的人会理解对方提问的目的，然后提供有用的信息，而不是按照自己的意思解释字面上的问题。
基层的员工要抬起头，而管理者要弯下腰。
